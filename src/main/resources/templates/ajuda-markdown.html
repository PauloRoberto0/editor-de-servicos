O que é Markdown?

Markdown é um formato de texto leve e fácil de usar para estilizar todos os tipos de escrita. Você controla a forma como o documento é exibido; criar listas e incluir links no texto são apenas algumas das coisas que podemos fazer com Markdown.

Em sua maioria, Markdown é apenas texto simples, com alguns caracteres que não fazem parte do alfabeto para ditar como a exibição será feita, como # ou *.

Itálico

Para demarcar texto que deve ser exibido em itálico, coloque um asterisco antes e outro depois do texto. Por exemplo: 

    Este texto é comum, mas *este está em itálico*, como esperado.

Quando formatado pelo Markdown, o texto acima ficará assim:

Este texto é comum, mas este está em itálico, como esperado.

Negrito

Para demarcar texto que deve ser exibido em negrito, coloque dois asteriscos antes e outros dois após o texto. Por exemplo: 

    Este texto é comum, mas **este está em negrito**, como esperado.

Quando formatado pelo Markdown, o texto acima ficará assim:

Este texto é comum, mas este está em negrito, como esperado.

Listas

Com Markdown, é fácil criar listas, numeradas ou não. Para criar uma lista numerada, basta elencar um item por linha, começando com o número do mesmo, e um ponto (.), seguido do texto desejado: 


    1. Este é o primeiro item
    2. Este é o segundo
    3. E assim por diante...

Listas não-numeradas são ainda mais simples: elenque cada item em uma linha, começando-a por um asterisco (*) e um espaço, seguido do texto desejado: 


    * Este é o primeiro item
    * Este é o segundo
    * E assim por diante...

Links

Para incluir um link no texto, escreva o texto que deverá conter o link entre chaves, seguido pelo link entre parênteses. Por exemplo: 


    Este é um link para o [Portal de Serviços](http://servicos.gov.br).

Quando processado pelo Markdown, o link acima será exibido assim:

Este é um link para o Portal de Serviços.

Combinações

É possível combinar listas e links facilmente: 


        * [Portal de Serviços](http://servicos.gov.br)
        * [Editor de Serviços](http://servicos.gov.br/editar)
        * [Portal Brasil](http://brasil.gov.br)

Será exibido da seguinte forma:

  • Portal de Serviços
  • Editor de Serviços
  • Portal Brasil
