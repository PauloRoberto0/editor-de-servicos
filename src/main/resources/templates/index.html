Passo 2: Dados Obrigatórios

Título

Esse é o nome que aparecerá em destaque. O título deve ser curto e claro para o cidadão. Caso haja uma sigla do nome deste serviço, inclua-a entre parênteses ao final do título.

O que é?

Faça uma descrição sintética do serviço, com ênfase no valor para o cidadão. Ao ler, o cidadão saberá se este é mesmo o serviço que ele está procurando.

Quem pode utilizar esse serviço?

Descreva brevemente quem são as pessoas que satisfazem as condições para acessar este direito ou utilizar este serviço. Preferencialmente, organize em tópicos, como uma lista, começando sempre com um hífen “-“ e letra maiúscula, separando cada item com um “enter” (quebra de linha).

Resultados esperados

Descreva aqui os documentos gerados e outros resultados alcançados ao final dos procedimentos relacionados a este serviço. É fundamental que o texto deixe claro quais os direitos que o requerente passa a poder exercer.

O que é necessário para acessar este serviço?

Descreva quais documentos o cidadão deverá apresentar para acessar este serviço. Caso certos documentos sejam necessários apenas em alguns casos, é preciso deixar claras as condiçõesem que isso acontece. Caso haja múltiplos casos de acesso, com diferentes demandas por documentos, experimente organizar a informação em tópicos. Preferencialmente, organize em tópicos, como uma lista, começando sempre com um hífen “-“ e letra maiúscula, separando cada item com um “enter” (quebra de linha).

Passo-a-passo para a realização deste serviço

Descreva os principais passos que o cidadão deverá realizar para obter este serviço. Utilize apenas uma linha para cada passo, e seja breve. O objetivo é situar o cidadão sobre como proceder.

Próximo passo

Descreva qual o próximo passo que o cidadão deverá tomar para continuar utilizando este serviço. É importante que a informação sobre como proceder no processo dessa solicitação seja descrita de forma objetiva e organizada.

Tempo estimado

Descreva quanto tempo em média o cidadão levará para obter esse serviço.

Entre e dias.

Caso haja exceções à estimativa de tempo deste serviço, será mostrado um texto de esclarecimento ao cidadão.
